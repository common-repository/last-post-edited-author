== Description ==

This is a simple plugin that adds the last user/author of a page or post and the time and date of that edit to the Posts/Pages table in your Dashboard


Requires WordPress 3.0 + 

Get in touch with me at:
http://mattroyal.co.za/

**Features**

* Adds the last person to edit page or post to the dashboard page/post colomns

== Usage ==

* Enable the Plugin
* Thats all folks :-)

== Installation ==

Installing "Last Post Edited Author" can be done either by searching for "Last Post Edited Author" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
